JDMSystem need the following libraries:
---------------------------------------

QtJambi in version 4.4.3
  - example: qtjambi-4.4.3_01.jar (rename to qtjambi.jar)
  - example: qtjambi-win32-msvc2005-4.4.3_01.jar (rename to qtjambi-runtime.jar)

iBATIS for Java in version 2.3.4.726 
  - ibatis-2.3.4.726.jar (rename to ibatis.jar)

MySQL Connector/J in version 5.1.7
  - mysql-connector-java-5.1.7-bin.jar (rename to mysql.jar)
